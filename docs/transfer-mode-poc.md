# Transfer Mode POC

## What This Solves

紙面へ回答を転記する作業のために、通常の学習画面とは別に、転記対象だけを確認しやすい一覧表示を用意する。

今回の POC では、`完了` 済みの問題を現在の問題セットの全ページから抽出し、印刷しやすいレイアウトで表示する。

## Design

- 通常表示とは別に `転記モード` を持つ
- 転記対象は「現在の問題セットの全ページにあり、かつ `完了` 済みの問題」
- 表示内容は、問題の見出し、`入力内容` または `解答` の切替表示を基本にする
- 入れ子の小問は再帰的に拾い、各行を `ラベル / 問題文 / 図表 / 転記値` の順で表示する
- 数直線、表、ヒストグラムなどの図表は、対応する問題本体または小問の `visuals` と `answerVisuals` から転記モード用に再構成する
- 入力欄がない問題は `answer` データを代替表示する
- `転記を印刷` はブラウザの `window.print()` を使う簡易実装にする

## Implementation

更新対象ファイル:
- `app/index.html`
- `app/src/main.js`
- `app/src/styles/page.css`
- `README.md`
- `docs/engineering-notes.md`
- `docs/transfer-mode-poc.md`

実装上の要点:
- ツールバーに `転記モード` `転記内容` `転記を印刷` を追加する
- `state.transferMode` で通常表示と転記表示を切り替える
- 通常の `renderProblems()` とは別に、転記用カードを `main.js` 側で組み立てる
- 転記カードの中では、問題本体と入れ子小問の回答行を分け、各行ごとに必要な図表を再描画する
- 印刷時はツールバーを隠し、転記一覧だけを紙面向けに整える

## Operation

1. 回答を入力し、転記したい問題へ `完了` を付ける
2. 問題セットを選び、通常表示で必要な入力を進める
3. 上部の `転記モード` を押して一覧へ切り替える
4. 転記モードでは、現在の問題セットの全ページから `完了` 済み問題だけが抽出される
5. 必要に応じて `転記内容` で `入力内容` と `解答` を切り替える
6. 内容を確認し、`転記を印刷` で印刷または PDF 保存する

## Limitations

- 数直線、表、ヒストグラムは転記モードでも再描画するが、未対応の visual type が増えた場合は `main.js` 側の転記用組み立て処理を追加更新する必要がある
- `完了` は学習状態と転記対象を兼ねているため、将来的には `転記対象` や `転記済み` を別フラグに分ける余地がある
- 現在は「現在の問題セットの全ページ + completed」固定なので、複数 dataset を横断して一括抽出する機能はまだない
